## Data Pipeline & Methodology

Across the projects, a standardized machine learning pipeline was implemented to ensure rigorous and reproducible results:

1. **Exploratory Data Analysis (EDA):** Cleaning, normalizing, and visualizing raw datasets.
2. **Feature Engineering:** Selecting and transforming input variables to optimize algorithmic performance.
3. **Model Training:** Fitting data to statistical and algorithmic models using appropriate hyperparameters.
4. **Validation & Testing:** Evaluating model accuracy, precision, and recall against testing subsets to prevent overfitting.
